# ☁️ Event Management System (AWS Serverless)

A cloud-based event management system built using AWS serverless architecture. This application allows users to create events, register users, and manage event registrations efficiently.

---

## 🚀 Features

* Create and manage events
* Register users
* Register users for events
* View all events
* View event registrations

---

## 🛠️ Tech Stack

* AWS Lambda
* API Gateway
* DynamoDB
* HTML, CSS, JavaScript

---

## 📂 Project Structure

frontend/ → UI (HTML, CSS, JS)
backend/ → AWS Lambda functions

---

## 🔗 Architecture

Frontend → API Gateway → Lambda → DynamoDB

---

## 📌 Backend Functions

* createEvent
* getEvents
* registerUser
* registerForEvent
* getRegistrations
